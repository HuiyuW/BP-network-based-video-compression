##########Requirement##########
Install the Matlab Image Processing Toolbox, Deep Learning Toolbox.
The code is tested on Matlab R2020b, with operating system Windows.


##########Usage Description##########
Usage 1: Run the scripts
1. Unzip the file Huiyu_Wang_Optimization.
2. Open Matlab.
3. Click Browse for folder and add file to your workspace.
4. Double-click the two Optimizition functions, all functions are displayed here.
5. Enjoy and have fun!
